# AI-Email-Automation-Assistant


An AI-powered email automation workflow built using n8n, Google Gemini AI, and Gmail API.

## Features

- Gmail Trigger
- AI Email Classification
- Job Email Detection
- Meeting Email Detection
- AI-powered Reply Generation
- Automatic Gmail Draft Creation
- Professional Email Subject Generation
